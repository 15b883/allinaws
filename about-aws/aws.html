关于Amazon Web Service

Amazon Web Services（AWS）是Amazon的子公司，该公司以按需付费的方式为个人，公司和政府提供按需云计算平台和API。总体而言，这些云计算Web服务提供了一组原始的抽象技术基础架构以及分布式计算构建块和工具。这些服务之一是Amazon Elastic Compute Cloud，它允许用户随意使用虚拟计算机集群。 可通过Internet随时访问。AWS版本的虚拟计算机模拟了真实计算机的大多数属性，包括用于处理的硬件中央处理器（CPU）和图形处理单元（GPU），本地/ RAM内存，硬盘/ SSD存储，选择操作系统，联网，以及预先加载的应用软件，例如Web服务器，数据库，客户关系管理（CRM）等。

AWS 技术在全球的服务器场中实施，并由Amazon子公司维护。费用基于使用情况的组合，即“按需付费”模型，订户选择的硬件/操作系统/软件/网络功能，所需的可用性，冗余性，安全性和服务选项。订户可以为一台虚拟AWS计算机，一台专用物理计算机或两者的集群付费。作为订购协议的一部分，亚马逊为订购者的系统提供安全性。AWS在许多全球地理区域（包括北美的 6个）运营。

AWS包含了多种服务，服务范围广泛，包括计算，存储，网络，数据库，分析，应用程序服务，部署，管理，移动，开发人员工具和物联网工具。最受欢迎的包括Amazon Elastic Compute Cloud（EC2）和Amazon Simple Storage Service（Amazon S3）。大多数服务不是直接向最终用户提供，而是通过API提供功能供开发人员在其应用程序中使用。使用REST体系结构样式和SOAP协议，可以通过HTTP访问Amazon Web Services的产品。

与构建实际的物理服务器场相比，亚马逊向订户推销AWS是一种更快，更便宜地获得大规模计算能力的方式。[7]所有服务均根据使用情况计费，但每种服务以不同方式衡量使用情况。
