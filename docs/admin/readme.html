AWS入门小知识

首先，你要先注册一个AWS账号～.～

然后，

登录到aws控制台，打开IAM控制台，开启MFA安全认证，保证根用户的安全；

申请一个IAM用户并授权此用户最大权限（AWS架构师第一准册，不要使用根用户登录控制台操作）；

用申请的IAM用户进行所有的操作，根用户在没有必要的情况不要使用，并由专人负责；

熟悉AWS控制台，并在把控制台上面的服务选项都点击看看
